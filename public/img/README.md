# Images à fournir

Remplace ces fichiers par de vraies images (mêmes noms) :

- `kotel.jpg`            → hero, photo du Mur occidental (Kotel), Jérusalem (paysage, ~2000px)
- `guide-placeholder.jpg`→ avatar par défaut d'un guide (carré)
- `city-jerusalem.jpg`   → vignette ville (portrait 3:4)
- `city-tel_aviv.jpg`
- `city-dead_sea.jpg`
- `city-galilee.jpg`

Les photos des guides (champ `photo`) et les galeries pointent vers des URLs
externes (Unsplash dans le seed) — autorisées via `remotePatterns` dans next.config.ts.
